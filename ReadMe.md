### yummy-cli
一个很下饭的cli工具,一键式快速生成项目模板,自带路由/api请求/常用函数封装/常见需求代码建议.
让你少敲代码,多花点时间休息.

### 脚手架开发参考
参考文章：https://www.cnblogs.com/cangqinglang/p/10277977.html
参考脚手架工具: vue-cli 3.x

### 安装
`npm i yummy-cli -g`

### 使用
`yummy init`或者`y i`
